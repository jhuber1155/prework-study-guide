HTML

  • The meta tag helps us determine search engine algorithms
  • The title tag changes the top header on the browser
  • Linking src gives us a source for our images and the alt gives the search engine tags to look for
  • Headers come in various sizes from h1 to h6

CSS

  • We need to think of a website as a series of interlocking boxes. We use elements like margin (space on the outside) and padding (space on the inside) to seperate items
  • CSS attributes can be added in a seperate sheet which can be styled without having to add lines of code to each element of the HTML file.
  • We can style elements like color, text size, and background color.

Git

  • When using Git, be sure to check which branch you are working on before beginning by using "git status" as a command.
  • Use the "git bash command git checkout -b branch-name" to create a new branch and switch to it.
  • Be sure to save early and often to your head branch (also known as feature branch)
  • Read up on all the Git Bash commands you need to know to be effective in Git
